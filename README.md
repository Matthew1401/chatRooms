# chatRooms 💬

https://mateuszkonieczny.netlify.app/

## 🇵🇱 Wersja Polska

Stworzyłem prostą aplikację webową opartą na komunikacji w czasie rzeczywistym. Można ją przetestować, korzystając z podanego linku. Jeżeli po wciśnięciu przycisku »Submit« pojawi się komunikat: »Error: Server connection failed«, proszę odczekać parę sekund – serwer postawiony jest na darmowym hostingu i po pierwszym zapytaniu potrzebuje chwili na wybudzenie.

- Funkcje: Tworzenie własnych pokoi (nazwa, hasło, limit miejsc), publiczna lista wszystkich aktywnych pokoi, dostęp chroniony hasłem.
- Technologie: Vue.js, Socket.io (WebSocket), JavaScript, HTML, CSS.

## 🇬🇧 English Version

I've built a simple real-time web application. You can try it out using the link provided. If you get an 'Error: Server connection failed' message after clicking 'Submit', please wait a few seconds—the server is on a free hosting plan and needs a moment to spin up after the initial request.

- Features: Custom room creation (name, password, user limit), public directory of active rooms, password-protected access.
- Tech Stack: Vue.js, Socket.io (WebSocket), JavaScript, HTML, CSS.
